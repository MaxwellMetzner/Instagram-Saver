Instagram Saver

This extension is API-first. These settings control naming, confirmation, and metadata behavior without affecting how media is resolved.

Disables the popup on left click, starts the current page download directly, and shows a green badge when the page is downloadable or a red badge when it is not.
Shows a confirmation prompt summarizing the files about to be saved.
Writes a JSON file alongside the media with identifiers, timestamps, filenames, and source info.
Writes a small report file with pagination diagnostics, skipped items, and failed items for each download run.
Keeps sidecar JSON files out of the main media folder while still preserving them.
Best-effort comment export for individual posts and reels only. Profile batch downloads do not fetch comments.
Multi-file downloads always queue without prompting to avoid blocking the batch.
Controls whether prior downloads suppress repeated downloads across separate runs.
